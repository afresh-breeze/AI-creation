# 奶龙跑酷（nailong_run）

这是一个轻量 3D 跑酷小游戏：
你操控小球在赛道上左右移动，一边躲奶龙，一边吃积分，冲目标分数过关。

*谁不想急头白脸地来一局躲避奶龙呢！*

---

## 1) 游戏介绍

这是一个网页端 3D 跑酷小项目。

你会在持续前进的道路上完成几件事：
- 躲开奶龙障碍；
- 收集星光积分；
- 达到每关目标分数；
- 撑得越久，关卡越高。

*其实你可以在快要一头撞死的时候选择暂停，然后选择重新开始，设计者已贴心地设计为从本关开始！*  
*（当然前提是你反应够快）*

简而言之：**简单上手，但很容易“再来一把”**。

---

## 2) 游戏玩法

- **移动端**：在屏幕下方左右滑动控制小球。
- **PC 端**：`A / ←` 左移，`D / →` 右移。
- **暂停**：`空格`（或点击屏幕上方任意位置）。
- **目标**：躲障碍、吃积分，达到当前关卡目标分数。

通关口诀：**躲奶龙，骚走位，勤刷分**。

---

## 3) 涉及技术

- **Three.js**：场景、相机、光照、材质、动画与模型渲染。
- **HTML**：页面结构和 UI 容器。
- **CSS**：界面样式、动效和响应式布局。
- **JavaScript**：主循环、输入控制、碰撞检测、关卡与分数逻辑。
- **GLTFLoader**：加载奶龙模型（`.glb`）。

---

## 4) 核心功能实现

- **场景初始化**
  - 初始化 `scene / camera / renderer`。
  - 创建道路、小球、光照和背景特效。

- **奶龙模型加载与缓存**
  - 使用 `GLTFLoader` 异步加载奶龙。
  - 首次加载后缓存模板，后续直接 `clone`，减少等待。

- **障碍与积分动态生成**
  - 按路段实时生成障碍和积分。
  - 根据关卡调节刷怪密度与积分节奏，保证难度递进。

- **碰撞与得分系统**
  - 小球与障碍命中后判负。
  - 小球与积分命中后加分并触发收集效果。

- **关卡与 UI 同步**
  - 达到目标分数自动进入下一关。
  - 实时更新分数、关卡、速度与提示面板。

## 6) 整体逻辑

项目运行流程：

1. 初始化游戏资源（场景、模型、UI）。
2. 玩家输入控制小球左右移动。
3. 赛道持续向前，动态刷障碍和积分。
4. 每帧进行碰撞判定：
   - 拥抱奶龙：游戏结束；
   - 收集积分：分数增加。
5. 分数达标后进入下一关。
6. 循环以上流程，直到失败或继续冲更高关。  
  
*是不是易如反掌！可是要忍住不和奶龙拥抱好像有点困难（？）*
